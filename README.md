Team Project

This project was developed collaboratively as part of a two-member team for the Database Management Systems (DBMS) course at Thapar Institute of Engineering and Technology.

Both team members actively contributed throughout the project lifecycle, including requirement analysis, database design, normalization, SQL and PL/SQL implementation, testing, debugging, and documentation. The project reflects a shared effort in building a comprehensive and well-structured database solution.
